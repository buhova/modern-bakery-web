# 🍞 Modern Bakery Web

[🌐 Live Demo](https://buhova.github.io/modern-bakery-web/)

A modern, responsive website for a bakery showcasing products, chefs, and multiple locations. Designed to provide an attractive, easy-to-navigate experience for customers on any device.

📌 About
Modern Bakery Web allows visitors to:
- Submit a contact form to request a callback
- Explore detailed information about the bakery
- Learn about the chefs and their specialties
- View information about other bakery locations
- Find the bakery on an interactive map
- Access social media pages through direct links
- Navigate seamlessly on desktop and mobile devices

🚀 Technologies Used
- HTML 5
- SCSS / CSS
- JavaScript
- Responsive design for mobile and desktop
